# competition_cpp
The workspace on VSCode for the C++ user coding in a programming competition

## directories
- lib/
- src/
    - bin/
- b.sh
- br.sh
- r.sh

## use
code program in  
    `src/{run1.cpp, ..., run6.cpp}`  
and use
    `./br.sh {$1: 1, ..., 6}` to build and run  
    `./b.sh {$1: 1, ..., 6}` to build  
    `./r.sh {$1: 1, ..., 6}` to run
